# Taanya Tapke — Portfolio Website

## 🚀 How to deploy (5 steps)

1. **Create your GitHub repo** named exactly: `taanyatapke.github.io`
2. **Add these 3 files** to the repo: `index.html`, `style.css`, `script.js`
3. **Create an `assets/` folder** and add your `resume.pdf` inside it
4. **Go to Settings → Pages** in your GitHub repo → Source: Deploy from branch → main → Save
5. Visit `taanyatapke.github.io` — done! ✅

## 📁 File structure
```
taanyatapke.github.io/
├── index.html
├── style.css
├── script.js
└── assets/
    └── resume.pdf     ← add your resume here
```

## ✏️ Customising

- **Hero stats** — update the numbers in the `hero-stats` div in index.html
- **Projects** — add more `.project-card` divs inside `.projects-grid`
- **Accent colour** — change `--accent: #7c6bff` in style.css to any hex
- **Typed strings** — edit the `strings: [...]` array in script.js
- **LinkedIn link** — search and replace `taanyatapke` with your actual LinkedIn handle

## 🔗 Optional: Deploy to Vercel (faster)
Go to vercel.com → New Project → Import your GitHub repo → Deploy
Your site will auto-update on every git push.
